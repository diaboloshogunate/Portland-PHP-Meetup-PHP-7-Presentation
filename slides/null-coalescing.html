Null Coalescing - ??

Checks if the value is set and not null


// The old way
// triggers a E_NOTICE
$username = isset($_GET['user']) ?: 'nobody';
// to avoid the E_NOTICE it has to be written like this
$username = isset($_GET['user']) ? $_GET['user'] : 'nobody';

// With the ?? operator
$username = $_GET['user'] ?? 'nobody';
