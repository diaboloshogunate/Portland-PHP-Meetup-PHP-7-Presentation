Null Coalescing - ??

Syntactic sugar for setting default values.

Returns false if the value is not set or is null.


// The old way
$username = isset($_GET['user']) ?: 'nobody'; // triggers a E_NOTICE
$username = isset($_GET['user']) ? $_GET['user'] : 'nobody'; // to avoid the E_NOTICE

// With the ?? operator
$username = $_GET['user'] ?? 'nobody';
